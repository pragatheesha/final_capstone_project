# final_capstone_project
# End-to-End Customer Analytics & Churn Prediction System

## Project Overview
This project presents a complete end-to-end data science workflow designed to solve a real business problem: **customer churn prediction and customer segmentation**. The project integrates data preprocessing, exploratory analysis, feature engineering, machine learning, clustering, and business recommendation generation into one unified pipeline. The objective is to help businesses identify high-risk customers early and apply data-driven retention strategies.

## Business Problem
Customer churn directly impacts revenue and growth. Acquiring new customers is significantly more expensive than retaining existing ones.

This project answers:

- Why do customers churn?
- Which customers are most at risk?
- How can the business reduce churn using data?

## Dataset
Main dataset:
- `customer_churn.csv` (customer behavior & billing data)

Supporting datasets:
- `sales_data.csv`
- `house_prices.csv`

- Python

- Matplotlib / Seaborn

## Key Results

- High monthly charges strongly increase churn risk
- New customers are the most likely to churn
- Long-term contracts significantly reduce churn
- Customer segmentation identified 3 actionable groups
- Random Forest achieved strong predictive performance

## Business Impact

This system enables:
- Early churn detection
- Segment-specific retention strategies
- Improved customer lifetime value
- Data-driven decision making
